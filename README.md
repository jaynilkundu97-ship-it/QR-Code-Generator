# QR Code Generator

This is a Python project that generates a QR code from user input.

## What this project does
- Takes text or URL from the user
- Generates a QR code image
- Saves it as a PNG file

## Technologies used
- Python
- qrcode library

## How to run the project

1. Install required library

2. Run the Python file

3. Enter text or URL when asked

QR code image will be generated.
